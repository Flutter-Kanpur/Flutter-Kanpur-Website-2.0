# Flutter Kanpur Website V2

Flutter Kanpur Website V2 is a scalable community platform built for the Flutter Kanpur developer ecosystem.

The platform is designed to support:

- developer community engagement
- events and workshops
- hackathons and coding contests
- speaker and organizer showcases
- community resources and blogs
- gallery and media management
- admin dashboard management
- responsive cross-platform experience
- future scalability for ecosystem expansion

---

# Tech Stack

## Frontend

- Next.js 16 (App Router)
- JavaScript
- React
- Tailwind CSS
- Zustand
- TanStack Query

---

## Backend & Database

- Supabase
- PostgreSQL
- Supabase Auth
- Row Level Security (RLS)
- Supabase Storage

---

# Architecture

This project follows:

- Modular Frontend Architecture
- Feature-First Folder Structure
- Scalable Component-Based Design
- Clean API Layer Structure
- Server State Management

---

# State Management

This project uses:

- TanStack Query → server state
- Zustand → global UI state
- useState → local component state
